# Wallet App - Take Home Assignment

## 🔧 Setup Instructions

1. Clone repo
2. Run: `flutter pub get`
3. Run: `flutter run`

## 🧪 Platforms
✅ Android  
✅ iOS  
✅ Web (optional, responsive)

## 🛠️ Assumptions
- WebSocket is simulated with Timer.
- Payout always succeeds.
- No backend, all in-memory.
